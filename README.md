# sql_pizza_sales_analysis
SQL Pizza Sales Analysis

A comprehensive SQL-based analysis of pizza sales data focused on business insights and performance metrics. This project explores order trends, revenue generation, customer purchasing patterns, popular pizza categories and sizes, hourly order distribution, and top-performing products. Advanced SQL techniques including JOINs, CTEs, aggregate functions, window functions, and revenue analysis are used to transform raw transactional data into actionable business insights.
